# IP Flag Info

**IP Flag Info** is a handy Chrome browser extension that instantly shows your public IP address and its geographical location details with a single click.

## Key Features
- **Country flag display** in the Chrome toolbar — visual indication of the region associated with your public IP.
- **Tooltip on hover** showing:
  - public IP address;
  - country code
- **Dynamic flag update** — the toolbar icon automatically changes to reflect the flag of the new country whenever your public IP changes.
- **Sound notification** — an audible alert is played when your detected country changes (can be disabled in settings).
